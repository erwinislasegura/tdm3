# Diccionario de tablas

- `users`: cuentas de acceso.
- `roles`, `permissions`, `role_permissions`, `user_roles`: RBAC completo.
- `organizations`, `organization_members`, `clubs`: estructura multi-organización.
- `people`, `players`, `coaches`, `referees`: modelo de personas con roles deportivos.
- `venues`, `venue_tables`: recintos y mesas.
- `tournaments`, `tournament_categories`, `tournament_modalities`, `tournament_phases`: configuración de torneos.
- `registrations`: inscripciones.
- `groups`, `group_members`: fase round robin.
- `matches`, `match_sets`: partidos y sets.
- `rankings`, `ranking_rules`, `ranking_movements`: motor de ranking.
- `notifications`, `documents`, `audit_logs`, `settings`: operación transversal.
